# AI-Memory-Layers

Curated resource list of projects, tools, and resources related to AI memory layers — systems that give AI agents and LLMs persistent, contextual, or structured memory capabilities.

## Format Rules

- **Single `README.md`** — all resources in one file. Never split across multiple files.
- **Sections alphabetical.** Entries within each section also alphabetical.
- **No table of contents** unless the number of sections makes navigation genuinely difficult.
- **No "Contributing" section** unless explicitly requested.
- Resources are user-provided only — do not auto-populate or guess resources.
- For GitHub resources, include shields.io badges (stars, last commit), language, and author.
- For non-GitHub links, use a simpler entry format without badges.
