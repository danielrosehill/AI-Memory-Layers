# AI Memory Layers

A curated collection of projects, tools, and resources related to AI memory layers — systems that give AI agents and LLMs persistent, contextual, or structured memory capabilities.

This includes vector-based memory stores, conversation history managers, knowledge graph memory, agent memory frameworks, and related infrastructure.

> **Note:** This is a point-in-time snapshot and may be updated over time. It is non-exhaustive and inclusion does not imply endorsement.
